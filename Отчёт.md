<li>Отсутствие пузырей/деформаций.</li>

</li>

</ul>
</li>
<li>
<p><strong>Адаптивная намотка (синхронизированная):</strong></p>
<ul>

<li>Алгоритм рассчитывает траекторию намотки на основе диаметра кабеля.</li>
<li>Обратная связь: Датчики натяжения (HBM U10) → коррекция усилия (15±2 Н) и передача сигнала для тонкой настройки скорости экструдера.</li>
</ul>
</li>
<li>
<p><strong>Учёт и оптимизация:</strong></p>
<ul>
<li>MES-система (SAP ME) собирает данные по всему циклу.</li>
<li>Обратная связь: Автоматическая корректировка “рецептов” для различных типов кабеля.</li>
</ul>
</li>
</ol>

<h4 id="критерии-выбора-типов-роботов">3.8.1. Критерии выбора типов роботов</h4>
<p>Выбор основан на технологических требованиях, физических параметрах операций и экономической эффективности:</p>
<ul>
<li>Грузоподъемность (вес сырья, барабанов).</li>
<li>Точность (контроль толщины изоляции ±0,02 мм).</li>
<li>Условия эксплуатации (пыль, температура, влажность).</li>
<li>Способность к синхронизации по сети с другим оборудованием.</li>
<li>Гибкость (перенастройка под разные типы кабеля).</li>
</ul>
<h4 id="типы-роботов-и-их-целевое-назначение">3.8.2. Типы роботов и их целевое назначение</h4>
<ol>
<li>
<p><strong>Тяжелые промышленные роботы (Намотка):</strong></p>
<ul>
<li>Грузоподъемность (150–300 кг): перемещение барабанов (до 200 кг).</li>
<li>Защита IP65: работа в запыленной среде.</li>
<li>Высокая скорость и точность позиционирования для синхронной работы.</li>
</ul>
</li>
<li>
<p><strong>Прецизионные роботы (6-осевые) (Контроль качества, протяжка):</strong></p>
<ul>
<li>Точность ±0,03–0,05 мм: для измерений толщины изоляции (±0,02 мм) и точного позиционирования.</li>
<li>Компактность: работа в ограниченном пространстве возле экструзионной головки.</li>
<li>Высокая скорость и плавность движения для синхронизации с линией.</li>
</ul>
</li>
</ol>
<h4 id="сравнение-с-другими-типами-роботов">3.8.3. Сравнение с другими типами роботов</h4>
<ul>
<li><strong>Коллаборативные роботы (cobots):</strong> Не подходят из-за низкой грузоподъемности, ограниченной скорости и недостаточной жесткости для синхронной работы в жестком контуре.</li>
<li><strong>Скара-роботы (Delta):</strong> Недостаточная грузоподъемность и дальность действия для намотки крупных барабанов.</li>
<li><strong>Мобильные роботы (AGV):</strong> Избыточны для статичной линии; не обеспечивают требуемой точности и синхронизации.</li>
</ul>
<p><strong>Вывод:</strong><br>
Предложенные типы роботов максимально соответствуют технологическим и экономическим требованиям процесса. <strong>Ключевым дополнительным критерием</strong> является их способность работать в составе единого синхронизированного контура управления, что обеспечивает основной экономический эффект от проекта.</p>

<li><strong>Устройства для намотки:</strong> адаптивные механизмы под разные диаметры барабанов.</li>
</ul>
<h3 id="требования-к-системе-управления">4.4. Требования к системе управления</h3>
<ul>
<li><strong>Интерфейс:</strong> HMI-панель на русском языке с возможностью выбора профилей (тип кабеля, материал).</li>
<li><strong>Отчетность:</strong> автоматическое формирование отчетов по OEE, браку и энергопотреблению.</li>
<li><strong>Интеграция:</strong> поддержка протоколов Modbus, OPC UA для подключения к SCADA-системе.</li>
<li><strong>Обмен данными:</strong> для учета брака и статистики должен осуществляться обмен данными с MES-подобной системой.</li>
</ul>
<h3 id="требования-к-безопасности">4.5. Требования к безопасности</h3>
<ul>
<li>Лазерные сканеры для остановки роботов при приближении персонала.</li>
<li>Аварийные кнопки на всех участках линии.</li>
<li>Соответствие стандартам ISO 13849 (безопасность оборудования).</li>
<li>Наличие блокирующих и сигнализирующих устройств, исключающих работу РТК при нахождении сотрудников в зоне работы подвижных устройств.</li>
<li>Оснащение аварийными кнопками останова на конвейерах и роботизированных модулях.</li>
</ul>
<h3 id="требования-к-эксплуатации-удобству-технического-обслуживания-и-ремонта">4.6. Требования к эксплуатации, удобству технического обслуживания и ремонта</h3>
<ul>
<li>Минимизация использования специнструментов для выполнения периодического технического обслуживания РТК.</li>
<li>Возможность быстрой блочно-модульной замены узлов при ремонте.</li>
<li>Возможность замены модулей (шнек, нагревательные элементы) без остановки линии.</li>
</ul>
<h3 id="шефмонтаж-обучение">4.7. Шефмонтаж, обучение</h3>
<ul>
<li>Поставщик обеспечивает пуско-наладку и обучение персонала.</li>
<li>Поставщик обязуется осуществить силами своих специалистов шефмонтаж, наладку, испытание и запуск оборудования в эксплуатацию. Завершением процесса монтажа считается безаварийная работа РТК в течение 5 смен с требуемой производительностью.</li>
<li>Обучение персонала эксплуатации и обслуживания оборудования производится в процессе монтажа и пуско-наладки на территории предприятия. Персонал должен быть обучен самостоятельным методам эксплуатации, настройки и ремонта РТК. Завершением процесса обучения считается успешное автономное выполнение всех тестовых операций персоналом.</li>
</ul>
<h3 id="требования-к-документации">4.8. Требования к документации</h3>
<p>Документация к РТК должна быть представлена в виде одного комплекта на электронном носителе и двух комплектов на бумажном носителе. Описание должно быть на русском языке, в системе измерения СИ и содержать:</p>
<ul>
<li>Спецификацию оборудования.</li>
<li>Паспорта на оборудование и его элементы.</li>
<li>Программу планового ТОиР.</li>
<li>Описание и параметры точек подключения оборудования к системам энергоснабжения, потребление энергоносителей.</li>
<li>Принципиальные электрические, пневматические схемы и схемы подключения систем управления.</li>
<li>Чертежи 3D. Габаритные чертежи оборудования.</li>
<li>Общие инструкции по монтажу.</li>
<li>Электрические схемы.</li>
<li>Инструкции по запуску, эксплуатации, обслуживанию и устранению неполадок с описанием параметров и экранов на русском языке.</li>
<li>Каталог запасных частей.</li>
<li>Сертификаты на комплектующие.</li>
<li>Сертификат соответствия ТР ТС.</li>
</ul>
